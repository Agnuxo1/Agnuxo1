## Active projects

### 🌊 [pixelflow](https://github.com/Agnuxo1/pixelflow) — GPU Texture Reservoir Computing
A 2-D GPU texture acts as a fixed random reservoir evolving under cellular-automaton
rules (Gray–Scott, wave equation, continuous Life). A linear readout is trained on
the frozen final state. Three backends: NumPy, OpenGL (moderngl), CUDA (cupy).

- **17.95× speedup** over CPU on batched CUDA (RTX 3090)
- MNIST accuracy 0.9281 vs. 0.9261 raw-pixel baseline
- Apache-2.0 · `pip install pixelflow-rc`
- [Live demo on HuggingFace](https://huggingface.co/spaces/Agnuxo/pixelflow)

### 🕸 [OpenCLAW](https://huggingface.co/spaces/Agnuxo/OpenCLAW-Agent) — Decentralised Agent Network
Multi-agent orchestration with federated node architecture. Agents collaborate
across independent HuggingFace Spaces with no central coordinator.

### 🔬 NEBULA / NeuroCHIMERA — Holographic Neural Substrates
Long-running research line exploring emergent computation in optical and holographic
neural representations. Demos live on HuggingFace Spaces.

### 📡 [P2PCLAW](https://huggingface.co/spaces/Agnuxo/P2PCLAW-Hive-Research-Network) — Peer-to-Peer Research Network
Distributed AI research network where autonomous agents propose, verify, and publish
scientific results across a hive of independent nodes.
