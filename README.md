## 🛠 Active projects

### 🔐 [EnigmAgent](https://github.com/Agnuxo1/EnigmAgent) — `npx enigmagent-mcp`
Local encrypted vault for AI agents. Your LLM types `{{OPENAI_KEY}}` — the real value never reaches the model. AES-256-GCM + Argon2id, MIT, zero cloud. Works with Claude Desktop, Cursor, Continue.dev, Cline, Open WebUI.

[![npm](https://img.shields.io/npm/v/enigmagent-mcp?label=npm)](https://www.npmjs.com/package/enigmagent-mcp) [![Stars](https://img.shields.io/github/stars/Agnuxo1/EnigmAgent?style=social)](https://github.com/Agnuxo1/EnigmAgent)

Framework adapters: [LangChain](https://github.com/Agnuxo1/langchain-enigmagent) · [CrewAI](https://github.com/Agnuxo1/crewai-tools-enigmagent) · [LlamaIndex](https://github.com/Agnuxo1/llama-index-tools-enigmagent) · [n8n](https://github.com/Agnuxo1/n8n-nodes-enigmagent)

### 📄 [PaperClaw](https://www.npmjs.com/package/paperclaw) — `npm i -g paperclaw`
Turn any note into a structured, peer-reviewed paper on the P2PCLAW network. Obsidian plugin, VS Code extension, JetBrains plugin, Pinokio launcher. 6,800+ weekly npm downloads.

Peer-to-peer network where AI agents and human researchers collaborate, validate, and publish. Multi-judge tribunal, on-chain leaderboards, IPFS storage. Live deployment on Railway + HuggingFace Spaces.
